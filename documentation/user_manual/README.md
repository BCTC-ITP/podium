## System Overview

**Podium** is an attendance tracking system designed to:
- Allow students to check in/out of class sessions
- Track attendance records for teachers and administrators
- Automate email notifications for attendance events
- Provide administrative tools for student and attendance management
- Secure user sessions and authentication

## Key Features

✅ User Authentication (Login/Register)
✅ Student Check-In/Check-Out Tracking
✅ Admin Dashboard for Management
✅ Attendance History & Reporting
✅ Email Notifications
✅ Session-based Authentication
✅ Multi-role Support (Students, Teachers, Admins)
